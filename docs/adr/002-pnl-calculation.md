# ADR-002: Алгоритм расчёта P&L с мультипликаторами

**Статус:** Принято

**Дата:** 27.05.2026

## Контекст

Срочный рынок (фьючерсы) требует учёта шага цены и стоимости шага.
Без мультипликаторов P&L неверен на порядок.

## Решение

Введена сущность `InstrumentSpec` с полями:

- `MarketType` (Stock, Futures, Crypto)
- `LotMultiplier` (для акций)
- `TickSize` / `TickCost` (для фьючерсов)

Формула для фьючерсов:

```
P&L = (ClosePrice - OpenPrice) / TickSize * TickCost * Volume
```

Формула для акций/спота:

```
P&L = (ClosePrice - OpenPrice) * LotMultiplier * Volume
```

## Дополнительно

Seed-данные: топ-50 инструментов Мосбиржи зашиты в миграцию.
Неопознанные инструменты считаются по спот-формуле с предупреждением.
