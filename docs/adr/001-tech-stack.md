# ADR-001: Выбор технологического стека

**Статус:** Принято

**Дата:** 27.05.2026

## Контекст

Требуется выбрать стек для десктоп-ориентированного веб-приложения с rich-интерфейсом,
профессиональными графиками и возможностью офлайн-работы.

## Решение

- **Backend:** .NET 8 + EF Core 8 + PostgreSQL 16 + Redis
- **Frontend:** React 18 + TypeScript + MUI v6 + lightweight-charts
- **Desktop:** Electron
- **Инфраструктура:** Docker Compose

## Обоснование

- PostgreSQL: ACID, GIN-индексы для тегов, оконные функции для Equity Curve
- MUI: готовая дизайн-система, настраиваемая тёмная тема
- lightweight-charts: производительность на больших выборках данных
- Electron: PWA-first, один код на веб и десктоп

## Альтернативы

- SQLite — отклонено (нет GIN, слабая конкурентность)
- Tauri — отклонено (незрелая экосистема для .NET)
